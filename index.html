Kingdom Realm SMP

Wiki Home

Kingdom Realm SMP is an advanced medieval kingdom simulation mod for Minecraft Forge 1.20.1+ featuring AI-driven kingdoms, complex diplomacy systems, seasonal events, and strategic warfare mechanics inspired by Indian Rajput history.

Getting Started

  1. Install Minecraft Forge 1.20.1 or higher
  2. Download Kingdom Realm SMP mod from Modrinth or CurseForge
  3. Place the .jar file in your mods folder
  4. Launch Minecraft and create a new world
  5. Use /kingdom help to view available commands
